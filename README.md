# How to use?
```
npm install
cd client && npm install

environment variables required
PROD_DB_STRING
DEV_DB_STRING
SECRET_KEY
REFRESH_SECRET_KEY
ACCESS_TOKEN_EXPIRES
REFRESH_TOKEN_EXPIRES

```
